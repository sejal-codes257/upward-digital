## Adding New Images

1. Drop the image file into this folder
2. Open `src/utils/images.js` and add it to the relevant array
3. Reference in `src/data/siteData.js` if needed

## Recommended Sizes
- Portfolio:  1200×800px  (3:2 ratio)
- Blog:       1200×675px  (16:9 ratio)
- Avatars:    200×200px   (1:1 ratio, will be circle-cropped)
- OG Image:   1200×630px
- Founder:    600×800px   (3:4 ratio)

Use WebP format for best performance. JPEG is also fine.
